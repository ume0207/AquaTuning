# Aqua Tuning — Talk prototype

Fincsのトーク画面をベースに、Aqua Tuningのスレッド型コミュニティとして再設計した静的プロトタイプです。

## 起動

`index.html` をブラウザで開くか、フォルダ内で簡易サーバーを起動します。

```bash
cd /Users/ume/Desktop/aquatuning-academy-portal
python3 -m http.server 4173
```

ブラウザで http://localhost:4173 を開いてください。

## 入っている機能

- スレッド一覧を目次のように表示
- 「トーク」「コンテンツ」「概要」の3カテゴリタブを維持
- 各カテゴリ内のスレッドを縦並びで選択
- スレッド選択後の詳細表示とスレッド内投稿
- いいねリアクション
- モバイル用ドロワーメニュー
- URLの `?thread=welcome` などによるスレッド直リンク
- スマホファーストの固定ヘッダー・下部ナビゲーション
- 投稿のブラウザ内保存
- 画像・検索・通知など、後続API接続用の操作フィードバック

本番化する場合は、認証、スレッド/投稿DB、コメント・通知API、権限管理を接続します。
